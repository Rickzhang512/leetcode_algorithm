## 目录结构

```
leetcode_algorithm/
├── java/             # Java代码目录
│   └── src/
│       ├── DataStructure/    # 数据结构实现
│       ├── leetcode/         # LeetCode题目解答
│       ├── Myself/           # 个人练习代码
│       └── Main.java         # Java主入口
│
└── python/           # Python代码目录
    └── src/
        ├── DataStructure/    # 数据结构实现
        ├── leetcode/         # LeetCode题目解答
        └── main.py           # Python主入口
```

## 说明

- **Java代码**：位于 `java/src/` 目录下
  - `DataStructure/` - 数据结构实现
  - `leetcode/` - LeetCode题目解答
  - `Myself/` - 个人练习代码
- **Python代码**：位于 `python/src/` 目录下
  - `DataStructure/` - 数据结构实现
  - `leetcode/` - LeetCode题目解答
- 两种语言的代码保持相同的目录结构，便于对照学习
